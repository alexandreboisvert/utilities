This is the ~/bin directory.

This directory contains many executable scripts.

nautilusCleanUpThumbnail.sh - Clean up the PNG files.
duplicates.py - If possible, I use fdupes (apt-get install fdupes),
    if it is not possible, this script will do the job.
py_compile.sh - Check the PEP8 and the syntax.
root.sh - Becoming root easily.
reboot1minute.sh - Reboots the machine in 1 minute.
shutdown1minute.sh - Shuts down the machine in 1 minute.
update_all.sh - apt-get update then apt-get upgrade in one command.
